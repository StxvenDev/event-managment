# Event API

1. Clonar los modulos del proyecto  ```yarn install```
2.  ```.env``` y configurar las variables.
   ```
    DB_DATABASE
    DB_USER
    DB_PASSWORD
   ```
3. Levantar la base de datos  ```docker-compose up -d```
4. Ejecutar en modo de desarrollo   ```npm start:dev```
